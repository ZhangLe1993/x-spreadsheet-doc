# loadData(data)

@param {data} json 数据格式

如何获取数据格式: 请使用 getData() 方法

  import Spreadsheet from "x-data-spreadsheet";

  const xs = new Spreadsheet("#x-spreadsheet-demo")

  // data 是 json 格式
  // 加载数据
  xs.loadData(data)
